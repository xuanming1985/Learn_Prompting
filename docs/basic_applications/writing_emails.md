# 🟢 写一封电子邮件

写电子邮件可能是一项耗时的任务，特别是当你需要先阅读一封你收到的电子邮件时。这个部分将涵盖从向你的老板发送简单的病假电子邮件，到更复杂的冷门营销邮件的使用案例。

## 基本的病假电子邮件

想象一下，你某天早上生病了，无法去上班（或者只是不想去 😈）。以下是一个简单的提示，写一封电子邮件给你的老板，告诉他们你病了。
